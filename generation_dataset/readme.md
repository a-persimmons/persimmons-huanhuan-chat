## 如何使用

1. 建议使用conda，使用 python 3.10
2. 安装依赖：`pip install -r requirements.txt`
3. 配置 [OpenAI_LLM.py](OpenAI_LLM.py)顶部 import 下面的 <api-key>和<model-api>
4. 修改 [main.py](main.py)中的“# CONFIG”部分
5. 启动: `python main.py`
